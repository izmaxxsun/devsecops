### List/Dictionary
- List comprehension 
```
new_list = [new_item for item in list if test]
```
- Dictionary comprehension
```
# iterating on list
new_dict = {new_key:new_value for item in list}

# iterating on dictionary
new_dict = {key:value for (key,value) in dict.items() if test}
```
